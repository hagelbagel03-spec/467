═══════════════════════════════════════════════════════════════
                        🆘 HILFE & SUPPORT
═══════════════════════════════════════════════════════════════

❓ HÄUFIGE PROBLEME:

🔧 "Node.js nicht gefunden":
   → Installieren Sie Node.js von https://nodejs.org/
   → Wählen Sie die LTS-Version
   → Starten Sie den Computer neu

🔧 "Build fehlgeschlagen":
   → Prüfen Sie Internetverbindung
   → Starten Sie als Administrator
   → Leeren Sie npm cache: npm cache clean --force

🔧 "App startet nicht":
   → Prüfen Sie Android-Version (min. 5.0)
   → Aktivieren Sie "Unbekannte Quellen"
   → Starten Sie Gerät neu

🔧 "Kein Internet in App":
   → Prüfen Sie WLAN/Mobile Daten
   → Firewall-Einstellungen prüfen
   → App-Berechtigungen kontrollieren
